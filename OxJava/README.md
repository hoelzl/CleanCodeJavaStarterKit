# OX Java

A starter kit for the "Ox" game (a variant of bulls and cows) that we use in
some of our trainings.

## Running from the Shell

To build the program and its tests:
```shell script
gradlew build
```

To run all tests:
```shell script
gradlew check
```

To build a jar archive (in `build/libs`):
```shell script
gradlew jar
```
